Meson の入手

Meson のダウンロード

Meson は GitHub リリースページ からダウンロードすることができます。

また Meson は Python パッケージインデックス に含まれるため、pip3 install meson によりインストールすることもできます。

開発版の最新は Git から直接入手してください。

依存パッケージ

Meson は Python 3 により実装されています。お使いのオペレーティングシステムにパッケージマネージャーがあれば、それを使って Python 3 をインストールすることができます。パッケージマネージャーがない場合は、Python のホームページ からダウンロードしてください。

利用しているプラットフォームや、バックエンドを利用するかどうかによって、Ninja モジュール を必要とすることもあります。ここでもできるだけ、ディストリビューションが提供しているモジュールを利用するようにしてください。それができない場合は、Ninja プロジェクトの Web サイトから入手してください。
