RPM モジュール

RPM モジュールは Meson 用の rpm スペックファイルのサンプルを生成するために用います。これはインストールされているファイルやその依存関係などを自動検出します。利用方法はいたって簡単です。Meson プロジェクトファイルの最終に (つまりトップレベルの meson.build ファイルの最後に) 以下の 2 行を追加するだけです。

rpm = import('rpm')
rpm.generate_spec_template()

一度 Meson を実行すると、ビルドディレクトリ内にテンプレートファイルが書き出されます。こうしたら上で書き加えた 2 行は削除します。そしてそのテンプレートファイルを編集し、不足する情報を書き加えます。こうしてこのファイルを利用します。
